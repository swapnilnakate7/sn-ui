# sn-card

A versatile card container component for Angular with header, body, and footer sections.

## Installation

```bash
npm i sn-card
```

## Usage

```typescript
import { ... } from 'sn-card';

@Component({
  imports: [SnCardComponent],
  template: `
    <sn-card title="Card Title">
      <p>Card body content goes here.</p>
    </sn-card>
  `
})
```
