# sn-spinner-x

A loading spinner component for Angular with multiple sizes and color schemes.

## Installation

```bash
npm i sn-spinner-x
```

## Usage

```typescript
import { SnSpinnerXComponent } from 'sn-spinner-x';

@Component({
  imports: [SnSpinnerXComponent],
  template: `
    <sn-spinner-x size="md" scheme="primary"></sn-spinner-x>
  `
})
```
