# SnUi - Angular UI Component Library

A comprehensive, modern UI component library built for **Angular 21.2.0**. SnUi provides a collection of reusable, customizable components designed to accelerate development and maintain consistency across your Angular applications.

## ğŸ“¦ Components

SnUi includes the following components:

- **sn-alert** - Alert notifications for user feedback
- **sn-badge** - Badge components for labels and status indicators
- **sn-button-x** - Enhanced button component with multiple variants
- **sn-card** - Card container for content grouping
- **sn-checkbox** - Checkbox input with indeterminate state support
- **sn-input** - Input field component with validation support
- **sn-modal** - Modal dialog component for user interactions
- **sn-radio** - Radio button component with grouping support
- **sn-select** - Dropdown select with search and multi-select
- **sn-spinner** - Loading spinner component
- **sn-table** - Data table with sorting and pagination
- **sn-tabs** - Tabbed interface for content organization
- **sn-textarea** - Multi-line text input with auto-resize
- **sn-toggle** - Toggle switch component

## ğŸš€ Quick Start

### Prerequisites

- **Node.js** - v20.0.0 or higher
- **Angular CLI** - v21.2.0
- **npm** or **yarn**

### Installation

```bash
npm install
```

### Development

Start the development server:

```bash
npm start
```

The application will be available at `http://localhost:4200/`

### Build

Build the project:

```bash
npm run build
```

### Testing

Run unit tests:

```bash
npm test
```

### Watch Mode

Build libraries in watch mode:

```bash
npm run watch
```
